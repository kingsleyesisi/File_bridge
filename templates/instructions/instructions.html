How to Use FileBridge

Follow these simple steps to start sharing files securely with anyone, anywhere.

1

Create or Join a Room

Click "Start File Transfer" and enter your name and a room name. You can create a new room or join an existing one by using the same room name.

Tip: Share the room name with others so they can join the same room and receive your files.

2

Share Files and Chat

Once in a room, you can send messages and share files up to 50MB. Click the paperclip icon to select files, or type messages in the text input.

Supported: Documents, images, audio, video, archives, and more file types up to 50MB.

3

Download Shared Files

When someone shares a file in your room, you'll see a download button. Click it to save the file to your device. Files are temporarily stored for easy access.

Note: Files are temporarily stored and may be removed after a period of inactivity.

Frequently Asked Questions

What's the maximum file size I can share?

You can share files up to 50MB in size. This covers most documents, images, and smaller media files.

How long are files stored?

Files are temporarily stored to allow room members to download them. Files may be removed after a period of inactivity to manage storage space.

Is my data secure?

Yes! We use industry-standard security practices to protect your files and conversations. Files are encrypted during transfer and storage.

Can I use FileBridge on mobile devices?

Absolutely! FileBridge is fully responsive and works great on smartphones, tablets, and desktop computers.
